## How to install
Install package yang dibutuhkan dengan menjalankannya pada terminal
```shell 
npm ci
```

Buat database Mysql pada PC Anda dengan nama database sesuai pada file lib/db.js

Jalankan website dengan menjalankannya pada terminal
```shell
npm start
```
